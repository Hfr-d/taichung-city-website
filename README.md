## 如何使用

### 方法一：直接開啟
1. 在檔案總管中找到專案資料夾
2. 雙擊 `index.html` 檔案
3. 網站將在預設瀏覽器中開啟

### 方法二：使用本地伺服器（建議）
```bash
# 使用 Python 3
python -m http.server 8000

# 使用 Node.js (需安裝 http-server)
npx http-server -p 8000

# 使用 PHP
php -S localhost:8000
```

然後在瀏覽器中開啟：`http://localhost:8000`

## 自訂與擴展

### 修改顏色
在 `style.css` 的 `:root` 區塊中修改 CSS 變數：

```css
:root {
  --primary-blue: #1E40AF;  /* 修改主色 */
  --accent-green: #10B981;  /* 修改輔助色 */
  /* ... 更多顏色設定 */
}
```

### 新增服務卡片
在 `index.html` 中複製現有的服務卡片結構：

```html
<div class="service-card">
    <div class="service-icon green">📝</div>
    <h3 class="service-title">服務名稱</h3>
    <p class="service-description">服務說明</p>
</div>
```

### 修改內容
直接編輯 `index.html` 中的文字內容即可。
